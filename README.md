# Profile Dashboard

A small dashboard application for displaying user profile data and posts with filtering capabilities.

## Live Demo
https://profile-dashboard-n7wm4q9ix-sedno-nstds-projects.vercel.app/

## Repository
https://github.com/sedno-nstd/profile-dashboard

## Tech Stack
- Next.js
- TypeScript
- React
- React DatePicker
- CSS

## Features
- User profile header with stats
- Posts feed with two display modes
- Date range filtering (from / to)
- View post engagement data at a selected time (likes & comments)
- Load more pagination
